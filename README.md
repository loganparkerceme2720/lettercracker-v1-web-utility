## Overview

LetterCracker is a compact web-based helper for tackling letter puzzles inside the browser. It focuses on anagrams, cryptograms, and related word-game problems where fast combination checks and quick results are useful.

Since it ships as a browser-ready, offline-capable single-file HTML app, there is no setup process to get in the way. That makes it convenient for puzzle fans, casual solvers, and anyone who wants a straightforward letter-cracking tool on desktop or mobile.

## Getting started

There is no traditional installation step.

1. Download or clone the repository.
2. Open the main HTML file in a modern web browser.
3. Begin entering letters or custom groups to test combinations.

If you want to host it yourself, copy the files to any static web server and open the app in your browser.

## How to use it

Launch the app and type in the letters you want to work with. LetterCracker will build and evaluate combinations so you can quickly narrow down possible words or patterns.

A common workflow looks like this:

1. Enter a letter set or custom group.
2. Inspect the combinations that are generated.
3. Update the input as the puzzle changes.
4. Keep iterating until a helpful match appears.

For offline use, open the HTML file directly from a local folder or device.
